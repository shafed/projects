# Сортировка массивов методом стандартного обмена

## ВВЕДЕНИЕ

В настоящее время крайне актуальна проблема эффективной обработки и
упорядочивания больших объемов данных. С ростом информационных потоков в
современных системах управления и программных комплексах задача поиска
оптимальных способов организации данных становится фундаментальной. Согласно
классификации алгоритмов, сортировка является одной из базовых операций,
напрямую влияющей на быстродействие и ресурсоемкость программного обеспечения.

На сегодняшний день реализовано множество алгоритмов сортировки: от простых
квадратичных до сложных логарифмических, таких как QuickSort или MergeSort. В
системном программировании выбор конкретного метода часто зависит от ограничений
памяти и специфики оборудования. Алгоритм сортировки методом стандартного обмена
(или «пузырьковая сортировка») является классическим примером, позволяющим
наглядно продемонстрировать принципы работы с массивами и оценки вычислительной
сложности.

Наиболее важным аспектом при разработке низкоуровневого программного обеспечения
является контроль за использованием ресурсов ЭВМ. Язык программирования Си
остается эталоном в этой области, так как он предоставляет прямой доступ к
памяти и позволяет реализовать алгоритмы с максимальной прозрачностью.
Использование Си для реализации сортировки массивов объемом 1000, 5000 и 10000
элементов позволяет оценить реальную эффективность алгоритма в зависимости от
масштаба входных данных.

Базовой функцией любой информационной системы является функция структурирования
данных. В данной курсовой работе необходимо реализовать алгоритм сортировки
методом стандартного обмена, причем необходимо обеспечить контроль корректности
работы на различных объемах данных, выполняя вывод контрольных элементов из
начала, середины и конца массивов.

Целью работы программная реализация системы сортировки массивов методом
стандартного обмена на языке Си.

Для достижения поставленной цели необходимо решить следующие задачи:

1. Изучить теоретические основы и сложность алгоритма «пузырьковой сортировки».
2. Разработать алгоритмическое обеспечение, включая блок-схемы программы и
   функций.
3. Реализовать программу на языке Си.
4. Провести тестирование на массивах заданного объема (1000, 5000, 10000
   элементов) и проанализировать результаты.

Объектом исследования выступают алгоритмы внутренней сортировки одномерных
массивов данных. Предметом исследования является программная реализация
алгоритма стандартного обмена на языке Си. Методы исследования: анализ
алгоритмической сложности, структурное программирование, эмпирическое
тестирование.

Структура расчетно-пояснительной записки: работа состоит из введения, трех
разделов основной части, заключения, списка использованных источников и
приложений. В первом разделе проводится анализ задания и выбор средств
разработки; во втором разрабатывается алгоритмическое обеспечение; в третьем
описывается программная реализация и результаты тестирования.

## 1. АНАЛИЗ ПОСТАВЛЕННОЙ ЗАДАЧИ

Основной задачей является создание программного обеспечения на языке **Си** для
автоматизации процесса упорядочивания данных. Программа должна реализовывать
следующие функции:

- **Генерация исходных данных:** формирование массивов размерностью **1000, 5000
  и 10000 элементов**. Для заполнения массивов будет использоваться функция
  генерации случайных чисел `rand()`.
- **Алгоритмическая обработка:** реализация классического алгоритма **сортировки
  методом стандартного обмена** (пузырьковая сортировка).
- **Контроль результатов:** организация вывода на экран по **10 элементов из
  начала, середины и конца** каждого массива до и после выполнения сортировки.

Для разработки выбран **язык программирования Си**. Использование стандартных
библиотек `<stdio.h>` и `<stdlib.h>` позволяет обеспечить ввод-вывод данных и
генерацию случайных чисел.

## 2. АЛГОРИТМИЧЕСКОЕ ОБЕСПЕЧЕНИЕ

Программа содержит следующую структуру:

1.  **Главная функция (`main`):** отвечает за управление логикой программы,
    объявление массивов заданных размеров, вызов функций вывода и сортировки.
2.  **Функция сортировки (`bubbleSort`):** функция, реализующая сортировку.
3.  **Функция вывода массивов (`printArray`):** функция вывода элементов массива
    , что повышает читаемость и структурированность кода.

- **Принцип работы:** последовательно сравниваются соседние элементы $arr[j]$ и
  $arr[j+1]$. Если порядок в паре неверный ($arr[j] > arr[j+1]$), элементы
  меняются местами.
- **Сложность:** временная сложность алгоритма составляет $O(n^2)$, так как в
  худшем случае (массив отсортирован по убыванию) требуется произвести
  $n(n-1)/2$ сравнений и обменов.

| Имя переменной | Тип данных | Назначение                                             | Область видимости         | Объем памяти (байт)\* |
| :------------- | :--------- | :----------------------------------------------------- | :------------------------ | :-------------------- |
| `sizes`        | `int[]`    | Массив с заданными объемами данных (1000, 5000, 10000) | `main`                    | 12                    |
| `n`            | `int`      | Текущий размер обрабатываемого массива                 | `main`, параметры функций | 4                     |
| `arr`          | `int[]`    | Основной массив элементов для сортировки               | `main`, параметры функций | $n \times 4$          |
| `i`, `j`       | `int`      | Индексы (счетчики) для организации циклов              | Локальная (в циклах)      | по 4                  |
| `temp`         | `int`      | Вспомогательная переменная для обмена элементов        | `bubbleSort`              | 4                     |

Функция `printArray` реализует вывод 10 элементов из начала, середины и конца
массивов

## ПРИЛОЖЕНИЕ Б

Листинг исходного кода программы main.c

```c
#include <stdio.h>
#include <stdlib.h>

void printArray(int arr[], int n) { // Функция для вывода массивов
  // Выводим 10 элементов из начала
  printf("\nВ начале: ");
  for (int i = 0; i < 10; i++)
    printf("%d ", arr[i]);

  // Разделитель вывода
  printf("\n");
  for (int i = 0; i < 80; i++)
    printf("—");
  printf("\n");

  // Выводим 10 элементов из середины
  printf("\nВ середине: ");
  for (int i = n / 2; i < n / 2 + 10; i++)
    printf("%d ", arr[i]);

  printf("\n");
  for (int i = 0; i < 80; i++)
    printf("—");
  printf("\n");

  // Выводим 10 элементов из конца
  printf("\nВ конце: ");
  for (int i = n - 10; i < n; i++)
    printf("%d ", arr[i]);

  printf("\n");
  for (int i = 0; i < 80; i++)
    printf("=");
  printf("\n");
}

void bubbleSort(int arr[], int n) { // Функция для сортировки
  for (int i = 0; i < n - 1; i++) {
    for (int j = 0; j < n - i - 1; j++) {
      if (arr[j] > arr[j + 1]) {
        int temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
      }
    }
  }
}

int main() {
  int sizes[] = {1000, 5000, 10000}; // Массив размеров

  for (int i = 0; i < 3; i++) { // Для каждого размера
    int n = sizes[i];
    int arr[n];

    for (int j = 0; j < n; j++) // Заполняем массив
      arr[j] = rand();

    printf("\nМассив размера %i до сортировки:\n", n);
    printArray(arr, n); // Выводим до сортировки

    bubbleSort(arr, n); // Сортируем

    printf("\nМассив размера %i после сортировки:\n", n);
    printArray(arr, n); // Выводим после сортировки
  }

  return 0;
}
```
